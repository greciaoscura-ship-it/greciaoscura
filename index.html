Términos de Servicio - Grecia Oscura Pipeline

Última actualización: 25 de diciembre de 2024

1. Aceptación de los Términos

Al utilizar el sistema automatizado de publicación de contenido "Grecia Oscura Pipeline" (en adelante, "el Sistema"), usted acepta estos términos de servicio. Si no está de acuerdo con estos términos, no utilice el Sistema.

2. Descripción del Servicio

El Sistema es una herramienta automatizada diseñada para gestionar y publicar contenido educativo sobre mitología griega en diversas plataformas de redes sociales, incluyendo YouTube, Instagram, Facebook y TikTok.

3. Uso del Servicio

3.1 Uso Permitido

  • Publicar contenido educativo sobre mitología griega
  • Gestionar publicaciones programadas en plataformas de redes sociales
  • Analizar métricas y engagement del contenido publicado

3.2 Uso Prohibido

  • Publicar contenido que viole los términos de servicio de las plataformas de redes sociales
  • Publicar contenido ilegal, difamatorio, o que infrinja derechos de terceros
  • Utilizar el Sistema para spam o actividades fraudulentas
  • Intentar acceder no autorizado a sistemas o datos

4. Contenido

4.1 Responsabilidad del Usuario

El usuario es responsable de todo el contenido publicado a través del Sistema. El usuario garantiza que:

  • Tiene los derechos necesarios sobre el contenido
  • El contenido no infringe derechos de terceros
  • El contenido cumple con las leyes aplicables y los términos de servicio de las plataformas

4.2 Contenido Educativo

El Sistema está diseñado específicamente para publicar contenido educativo sobre mitología griega. El usuario se compromete a mantener este enfoque educativo.

5. Plataformas de Redes Sociales

El Sistema se integra con plataformas de terceros (YouTube, Instagram, Facebook, TikTok). El usuario:

  • Debe cumplir con los términos de servicio de cada plataforma
  • Es responsable de mantener sus credenciales de acceso seguras
  • Reconoce que cada plataforma puede modificar sus APIs y términos en cualquier momento

6. Disponibilidad del Servicio

El Sistema se proporciona "tal cual" y "según disponibilidad". No garantizamos:

  • Disponibilidad ininterrumpida del servicio
  • Ausencia de errores o interrupciones
  • Compatibilidad con futuras versiones de las APIs de las plataformas

7. Limitación de Responsabilidad

En la máxima medida permitida por la ley:

  • El Sistema se proporciona sin garantías de ningún tipo
  • No seremos responsables por daños directos, indirectos, incidentales o consecuentes
  • No seremos responsables por pérdidas de datos, ingresos o oportunidades comerciales

8. Modificaciones

Nos reservamos el derecho de:

  • Modificar estos términos de servicio en cualquier momento
  • Modificar o discontinuar el Sistema sin previo aviso
  • Actualizar la documentación y funcionalidades

9. Privacidad

El Sistema procesa datos necesarios para su funcionamiento, incluyendo:

  • Credenciales de acceso a plataformas de redes sociales (almacenadas localmente)
  • Métricas y analytics de contenido publicado
  • Configuraciones y preferencias del usuario

Todos los datos se almacenan localmente en el sistema del usuario y no se transmiten a servidores externos, excepto a las plataformas de redes sociales para las cuales el usuario ha autorizado el acceso.

10. Propiedad Intelectual

11. Terminación

El usuario puede dejar de utilizar el Sistema en cualquier momento. Nos reservamos el derecho de suspender o terminar el acceso si se violan estos términos.

12. Ley Aplicable

Estos términos se rigen por las leyes del país donde se utiliza el Sistema.
